# Правила дизайна — Hostel Manager

Дизайн вдохновлён системой Airbnb: сдержанность, много воздуха, один
акцентный цвет. Интерфейс рассчитан на нетехнических пользователей —
крупные элементы, минимум на экране, понятный язык.

## Шрифт

- **Manrope** (Google Fonts) — геометрический гротеск, близкий аналог
  фирменного Airbnb Cereal. Поддерживает кириллицу.
- Заголовки — вес **600** (semibold). Не использовать 700+.
- Основной текст — вес 400, базовый размер 16px.
- Вторичный текст / подписи — 14px, цвет `muted-foreground`.

## Цвета

| Назначение            | Значение  | Где использовать                          |
| --------------------- | --------- | ----------------------------------------- |
| Акцент (Rausch)       | `#FF385C` | Только главные кнопки-действия и редкие акценты |
| Текст (ink)           | `#222222` | Заголовки и основной текст                 |
| Вторичный текст       | `#717171` | Подписи, подсказки                         |
| Фон                   | `#FFFFFF` | Фон страниц                                |
| Подложка              | `#F7F7F7` | Вложенные блоки, шапки таблиц               |
| Граница               | `#DDDDDD` | Рамки карточек, полей, таблиц               |
| Ошибка                | `#C13515` | Сообщения об ошибках, удаление              |

**Главное правило:** страница на ~90% белая + тёмный текст. Акцентный
коралловый цвет — 1–2 точки на экран, не больше.

## Скругления

- Базовый радиус — **14px**: карточки, модальные окна, поля ввода, кнопки.
- Вкладки и небольшие «пилюли» — полное скругление (`rounded-full`).

## Кнопки

- **Главная (CTA)** — сплошная коралловая, белый текст. Одна на блок.
- **Второстепенная** — контурная: граница, белый фон, тёмный текст.
- Высота крупная (40–48px) — удобно нажимать.

## Отступы

- Сетка 4 / 8px. Между крупными блоками — 20–24px, внутри карточек — 16–20px.

## Компоненты

- **Карточки** — белые, граница `#DDDDDD`, радиус 14px, без тяжёлых теней.
- **Модальные окна** — белые, радиус 14px, затемнение фона.
- **Активная вкладка** — тёмная заливка (ink), форма пилюли.
- **Таблица-шахматка** — шапка на подложке, сегодняшний день отмечен акцентом.

## Токены

Все цвета и радиус заданы CSS-переменными в `src/app/globals.css`
(`--primary`, `--foreground`, `--muted`, `--radius` и т.д.). В коде
использовать семантические классы Tailwind (`bg-primary`, `text-foreground`,
`text-muted-foreground`, `rounded-lg`), а не жёстко заданные цвета.
